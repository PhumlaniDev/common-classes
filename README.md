# common-classes

Small Java library containing commonly used classes and utilities for JVM projects.

## Repository
- Branch: `main`
- Remote: `git@github.com:PhumlaniDev/common-classes.git`

## Features
- Reusable domain and utility classes
- Maven-based build and test lifecycle
- Designed to be used as a dependency in other projects

## Requirements
- Java: JDK 21\+
- Build tool: Maven 3.9.9\+

## Build
```bash
mvn clean install
```
## Test

```bash
mvn test
```

## Usage
Add the library as a Maven dependency (replace coordinates with actual values):
```xml
<dependency>
  <groupId>com.example</groupId>
  <artifactId>common-classes</artifactId>
  <version>0.0.1-SNAPSHOT</version>
</dependency>
```
